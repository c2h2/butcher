## README
Manage words library from web ui.

## Get started:
  cd www
  bundle
  bundle exec rake db:create
  bundle exec rails s
  # Visit http://localhost:3000/

## Ruby version

ruby 1.9.3


## System dependencies

* Mongo database 2.2
* rails 4.0.0.rc1

## Configuration
Nothing need to do.

## Database creation
    cd www; bundle exec rake db:create




Please feel free to use a different markup language if you do not plan to run
<tt>rake doc:app</tt>.
